## 🚀 Execution

1. **Install Dependencies**:
   ```bash
   uv sync
   cd frontend && npm install
   ```

2. **Start Backend**:
   ```bash
   uv run python -m backend.main
   ```

3. **Start Frontend**:
   ```bash
   cd frontend && npm run dev
   ```
